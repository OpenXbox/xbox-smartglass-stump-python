Library to handle SmartGlass Stump (TV Streaming / IR control) messages.

For in-depth information, check out the documentation: (https://openxbox.github.io)

Dependencies
------------
* Python >= 3.6
* xbox-smartglass-core


How to use
----------

Install::

  pip install xbox-smartglass-stump
